Tabela

No html existem dois tipos de listas: Ordered lists (listas ordenadas) = ol unordered lists (listas não ordenadas) = ul dentro de cada uma dos termos de listas, temos os itens da lista List Itens=

  • ########################################## Exemplos=

    Menu Principal

    1. Home
    2. Especificações
    3. Fotos
    4. Mutimídia
    5. Fale conosco
    Resultado : 1.Home 2.Especificações 3.Fotos 4.Multimidia 5.Fale conosco ###################################################

    Menu Principal

    1. Home
    2. Especificações
    3. Fotos
    4. Mutimídia
    5. Fale conosco
    Resultado : a.Home b.Especificações c.Fotos d.Multimidia e.Fale conosco ops* caso coloque em "A" na tag type, as letras ficarão maiúsculas. #########################################################

    Menu Principal

    1. Home
    2. Especificações
    3. Fotos
    4. Multimídia
    5. Fale conosco
    Resultado : i.Home ii.Especificações iii.Fotos iv.Multimidia v.Fale conosco Caso for com "I", os numeros romanos serão em letras maiusculas também. ################################################### Caso eu queria uma lista ordenada começando do numero 3 :

    Menu Principal

    1. Home
    2. Especificações
    3. Fotos
    4. Mutimídia
    5. Fale conosco
    Resultado : 3.Home 4.Especificações 5.Fotos 6.Multimidia 7.Fale conosco ops* funciona do mesmo jeito com as letras, porém na tag "start" deverá ser em números. EX:

    Menu Principal

    1. Home
    2. Especificações
    3. Fotos
    4. Mutimídia
    5. Fale conosco
    Resultado : c.Home d.Especificações e.Fotos f.Multimidia g.Fale conosco ops* funciona do mesmo modo para números romanos. ----------------------------------------------------------------------------------------------------------- LISTAS NÃO ORDENADAS:

    Resultado : *.Home *.Especificações *.Fotos *.Multimidia *.Fale conosco valores para o type de ul : square: quadrados circle: circulo com fundo branco disc: circulo todo preto --------------------------------------------------------------------------------------------------------- LISTAS DENTRO DE LISTAS Posso fazer listas ul dentro de ol ou vice-versa.
