Nas versões anteriores ao HTML5, os estilos eram desenvolvidos por meio das tags da linguagem html. Porém, a partir dessa versão, os estilos são desenvolvidos através dos códigos da linguagem CSS. Como será exposto abaixo, a linguagem CSS é responsável pela formatação do estilo da página web.
