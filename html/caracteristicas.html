Características da Linguagem HTML

HTML é uma linguagem de marcação cuja principal finalidade é o desenvolvimento de websites. O termo marcação é devido a utilização dos “< > ” na execução dos códigos. Esses sinais são conhecidos como “tags”, que são as marcações que informamos para o documento em html para executarmos os comandos desejados, como criação de cabeçalhos, títulos, tabelas, entre outras funcionalidades.
