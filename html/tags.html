Tags

Tag é um código usado para marcar o início e, onde for requerido, o fim de um elemento HTML. Existem as tags de abertura e de fechamento. Uma tag de abertura é representada por sinal de menor ( < ), um nome de elemento HTML e um sinal de maior ( > ), que deve ser colocada imediatamente antes do início do conteúdo do elemento. Uma tag de fechamento se diferencia de uma tag de abertura apenas por uma barra ( / ) antes do nome do elemento . Ela deve ser colocada no final da construção do seu código. A combinação de tags e outros elementos textuais constituem a marcação. De modo resumido, podemos dizer que marcação é o conjunto de instruções que envolvem tags, textos, símbolos e outros elementos que possibilitam a geração das informações em um ambiente web.
