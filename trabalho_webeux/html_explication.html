1. O que é HTML

HTML (Hyper Text Markup Language é uma linguagem de marcação de texto muito utilizada para desenvolvimento de websites. Foi desenvolvida pensando na facilidade de o usuário entendê-la e conseguir desenvolver seus próprios sites. A linguagem HTML expressa informações também para sistemas e demais programas, através de tags específicas que atribuem valor semântico para o que se quer transmitir.

1.1 Características da linguagem HTML

HTML é uma linguagem de marcação cuja principal finalidade é o desenvolvimento de websites. O termo "marcação" é devido a utilização dos símbolos < > e </ > na execução dos seus códigos. Esses sinais são conhecidos como tags, que são as marcações que informamos para o documento em html para executarmos os comandos desejados, como criação de cabeçalhos, títulos, tabelas entre outras funcionalidades.

2. Tags ou marcações HTML

Tag é um código utilizado para marcar o início e, onde for requerido, o fim de um elemento HTML. Existem tags de abertura (< >) e de fechamento (</ >). Uma tag de abertura é representada por um sinal de menor (<) e de maior (>). A tag de fechamento se diferencia da tag de abertura apenas por conta de uma barra (/).

A combinação de tags e outros elementos textuais constituem a marcação. De modo resumido, podemos dizer que marcação é o conjunto de instruções que envolvem tags, textos, símbolos e outros elementos que possibilitam a geração das informações em um ambiente web.

Logo abaixo, temos um exemplo de como construímos algumas tags de abertura e fechamento em arquivo html.


<h3>Aqui está um exemplo de fechamendo de cabeçalho, identificado pela letra H 
(de header) seguido do número de hierarquia.</ h3>

<p>Esse é um exemplo de abertura e fechamento de um parágrafo.</ p>

<meta charset="utf-8"/>

<p>Podemos indicar o seu término dessa tag com uma (/) no final. 
Essa barra não é obrigatória.</ p>

3. Estruturas de um documento HTML

Todo arquivo em html é composto por os elementos: DOCTYPE, html, head, title e body, por padrão. Esses são as principais marcações que devem ser explicitadas em um documento html. Sem eles, o browser não conseguirá identificar o que estamos construindo no arquivo. Logo abaixo temos um modelo de como desenvolvemos essa estrutura em um arquivo html.


<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8"/>
	<title> título-da-pagina-html-no-navegador</ title>
</ head>
<body>
<h1>Cabeçalho</ h1>
<p>Parágrafo</ p>
</ body>
</ html>

Abaixo desse modelo, encontra-se uma tabela que explica cada tag utilizada.

Tags em um arquivo HTML

htmlIndica a raiz do arquivo em formato HTML
headIndica as estruturas de um documento HTML, como links, folhas de estilo ou metadados
metaDefine qualquer informação de metadados que não podem ser definidos por outros elementos HTML
titleDefine o título que será exibido na página do navegador
bodyNesta tag, informamos os dados do nosso website, como imagens, textos, vídeos e outros elementos

4. Marcadores de Estilo

Nas versões anteriores ao HTML5, as marcações de estilo da página html eram desenvolvidas em um mesmo arquivo html. Essa prática fazia com que o código ficasse muito longo, aumentando o tamanho do arquivo para a web, além de dificultar os trabalhos de revisão, sejam eles por terceiros ou pelo próprio desenvolvedor do programa.

A partir dessa versão, os estilos da página html, ou folhas de estilos, são desenvolvidos em outra linguagem: CSS. A linguagem CSS é responsável a partir da versão mencionada por toda formatação de estilo da página html. Ou seja, toda imagem, parágrafo, tabela e outros elementos de uma página web podem ser estilizados através do CSS.

No entanto, é possível estilizar a página web através da tag <style> no arquivo html. Abaixo será demonstrado uma versão de formatação das tags h2 e p em html e CSS, respectivamente.


Configurando o estilo de cabeçalhos em html:
<style>
	h2{
		color: blue;
		text-height: bolder;
	}
</ style>

Configurando o estilo de parágrafos em html:
<style>
	p{
		color: #dddddd;
		text-align: justify;
		text-indent: 20px;
	}
</ style>

Agora, vamos demonstrar como é feita a estilização acima, porém em CSS:


Configurando o estilo de cabeçalhos em CSS:
h2{
	color: blue;
	text-height: bolder;
}
Configurando o estilo de parágrafos em html:
p{
	color: #dddddd;
	text-align: justify;
	text-indent: 20px;
}

Importante destacar que, em CSS, conseguimos editar as mesmas tags em formatos diferentes. Na construção do nosso arquivo html, podemos definir as "id's" e "class" das nossas tags. Esses id's e class nos ajudam a formatar determinados campos do nosso arquivo.

Por isso, se for estilizar o seu site, faça uso da linguagem CSS. Adote também como boa prática utilizar um arquivo específico para seus estilos em CSS. Esse arquivo é comumente chamado de folha de estilo. Para carregar o seu arquivo, basta digitar o seguinte comando no campo head: <link rel="stylesheet" type="text/css" href="nome da pasta/nome do arquivo.css">

5. A estrutura head e header (cabeçalhos)

A tag head, assim como a tag header, significam cabeçalhos. No entanto, a tag head define os comportamentos globais do arquivo html, como importação de arquivos de estilo da página geral, definição da linguagem do arquivo, entre outras. Já a tag header define as propriedades de um determinado cabeçalho, onde são inseridos seus títulos, textos, imagens, menus entre outros elementos. Logo abaixo, temos um pedaço de código gerado em html para explicar melhor essa distinção.


<head>
	<meta charset="utf-8"/>
	<title> título-da-pagina-html-no-navegador</ title>
</ head>
<body>
	<div id="interface">
	<header id="principal">
		<h1>Cabeçalho</ h1>
		<p>Parágrafo</ p>
	</ header>
	</ div>
</ body>

6. Cores de texto e planos de fundo

Cores de texto. Pode ser alterada via css, com o comando “color”, exemplo: 

 
	<p id=”teste”> Texto desejado <p>

	p#teste {
	color: red;
	}

São utilizados para modificar o fundo de algum objeto no html, podendo alterar a cor, tamanho, posição entre outras funções. O “background” que é utilizado pelo css possui várias derivações, em que as principais são: 


div {

7 Imagens

Para inserir imagens via html, utilizaremos a tag < figure > e < img > com o atríbuto 'src' dentro de < img >, exemplo: 


< figure >

	<img src=”imagem.jpg”>   // src = source
