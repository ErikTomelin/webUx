7. Imagens

Para inserir imagens via html, utilizaremos a tag < figure > e < img > com o atríbuto 'src' dentro de < img >, exemplo: 


< figure >

	<img src=”imagem.jpg”>   // src = source

<⁄ figure >

// e se a img estiver contida dentro de uma pasta local, ficará assim: 

<img src=”pasta/imagem.jpg” >
obs* as imagens podem ser arquivos .jpg, .png, .gif

Além do atríbuto 'src', podemos também utilizar title="...", onde adiciona um comentário ao deixar o cursor sobre a imagem. 

 

< figure >

	<img src="pasta/imagem.png" title="texto desejado"> 

<⁄ figure >

7.2 Imagens flutuantes

Para uma imagem flutuar ao lado de um texto, utilizamos o comando "float". Exemplo: 

	<img  src="imagens/html4.png">

		CSS* 

	img{
		float:left;
		width: 100px
	}
